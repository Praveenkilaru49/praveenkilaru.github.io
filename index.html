Design and Development of a Low-Cost Tensile Testing Machine

This project focused on creating a compact, cost-effective tensile testing machine for academic use. It featured an Arduino Nano controller, load cell with HX711 amplifier, and NEMA 17 stepper motor. The machine precisely measured tensile forces and recorded real-time data during material elongation.

Role: CAD design, Arduino integration, load cell calibration, troubleshooting

Tools: SolidWorks, Arduino, HX711, NEMA 17, A4988, 3D Printing
